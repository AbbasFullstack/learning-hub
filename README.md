# 🎓 Learning Hub

A YouTube video search and player system. Users can search any topic and watch videos directly.

## Features
- 🔍 Search any topic (tutorials, courses, lessons)
- 🎬 YouTube videos fetch using YouTube API
- ▶️ Built-in video player (iframe embed)
- 📱 Fully responsive design
- 💡 Suggested search chips
- ⚡ Fast and lightweight

## Tech Stack
- HTML5, CSS3, JavaScript
- YouTube Data API v3
- Fetch API
